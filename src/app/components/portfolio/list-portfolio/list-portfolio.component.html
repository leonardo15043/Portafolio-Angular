Cine Colombia

En este proyecto se desarrollo un asistente cognitivo para automatizar la informacion de las peliculas y la venta de boleteria

Levantamiento de información

Se tuvieron varias sesiones de levantamiento para saber cómo implementariamos los diálogos y el flujo del asistente

Diseño de dialogos

Luego de tener los posibles escenarios de cada diálogo se montaron los flujos en IBM Watson assistant y se entreno el asistente diariamente

Sketch

Como la implementación era con el chat de facebook se construyeron algunos diseños basándose en los temas soportados por el API de Facebook

Desarrollo de Servicios

Se hizo una doble capa de servicios para que el asistente pudiera consumir la información, se pasó de un modelo SOAP a REST con microservicios en IBM cloud Functions para medir y optimizar la carga de información
